Fenômenos dos satélites de Júpiter

Botões de Tela

Configurar Data e Local

No calendário, a Data é alterada clicando-se sobre ela. O Ano é alterado clicando-se sobre o Mês. Os intervalos dos anos são alterados clicando-se sobre o Ano. Clicando-se nas setas esquerda ou direita é possível mudar os intervalos (décadas). Note que esses sucessivos cliques permitem ao usuário avançar ou regredir rapidamente no tempo. O modelo dinâmico utilizado nesse programa permite gerar os dados entre o ano de 2000 e 2100. Além de detectar as coordenadas geográficas locais automaticamente do computador, é possível introduzir dados manualmente ou escolhidos a partir de uma lista (busca textual).

Desligar Órbitas

Este botão permite desligar/ligar as linhas de representação das órbitas dos satélites de Júpiter.

Exportar Eventos

Opção para gerar os fenômenos dos satélites de Júpiter para o mês escolhido. O arquivo "satelites-jupiter.pdf" é gerado e automaticamente realizado uma cópia (download). Neste arquivo consta a data, instante observável, Azimute (Az) e Altura (H) de Júpiter, tipo de fenômeno, satélite envolvido no fenômeno e a duração dos eventos:

É importante notar que o Azimute (Az) pode ser usado como referência aproximada dos pontos cardeais. Quando esse ângulo for entre 315º e 45º, a posição estará aproximadamente na direção Norte. Já quando o ângulo estiver entre 45º e 135º, a posição estará aproximadamente na direção Leste. Para a direção Sul o ângulo de Azimute deve ser entre 135º e 225º e para a direção Oeste o ângulo está entre 225º e 315º.

  • Ocultações dos satélites por Júpiter
  • Trânsito dos satélites pelo disco de Júpiter
  • Eclipses dos satélites pela sombra de Júpiter
  • Trânsito da sombra dos satélites pelo disco de Júpiter

Nota: A duração do evento é exibida apenas para eventos que tem início e fim no mesmo dia devido a uma limitação de performance do programa.

Eventos do dia

O programa sempre inicia utilizando a data e hora local, com base no relógio do computador.

Passando o mouse sobre a ícone pode-se visualizar os resultados na tela. Apenas os resultados observáveis para a localidade configurada são exibidos.

Este botão mostra a trajetória dos satélites ao longo do mês escolhido pelo usuário.

Quando ativado, o botão permite visualizar este sistema de satélites de dois modos:

  • Vista da Terra
  • Vista Livre

Na opção Vista Livre chama-se a atenção para a possibilidade do sistema de Júpiter ser visto por vários ângulos e em fase, o que não é possível observar da Terra. Em outras palavras, a visualização funciona como se o observador estivesse ao redor de Júpiter.

A escala deslizante permite avançar ou recuar no tempo, dentro do período escolhido pelo usuário.

Os nomes dos quatro satélites galileanos IO, Europa, Ganimedes e Calisto são apresentados em cores, para que possam ser associados às linhas que descrevem graficamente os seus movimentos orbitais aparentes aproximados.

Funções do Mouse

  • Botão Esquerdo: com este botão pode-se deslocar (rotação) a imagem do sistema.
  • Botão Direito: para mover o plano do horizonte, sobre o qual está o observador, manter este o botão pressionado e arraste.
  • Botão de rolagem: o programa permite aproximar ou afastar a imagem, bastando para isso girar este botão.
