Ao clicar no botão "Plano ", o plano de referência ao qual este sistema de coordenadas celestes tem como base é visualizado. Para fins didáticos e de percepção de cada plano, pode-se ligar e desligar individualmente cada botão ou fazer uma combinação deles. Assim a visualização dos planos pode ser melhor compreendida.

Funções do Mouse

  • Botão Esquerdo: com este botão pode-se deslocar (rotação) a imagem do sistema.
  • Botão de rolagem: o programa permite aproximar ou afastar a imagem, bastando para isso girar este botão.
